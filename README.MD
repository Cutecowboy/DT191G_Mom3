# DT191G - Moment 3 

## Hemsidan
Hemsidan har två databas tabeller som innehåller följande:
1. Författarens namn och ålder
2. En boks titel och vem författaren är (kopplat till tabell 1)

Databaserna har en koppling emellan där en författare kan ha skrivit flera böcker.


## Uppgift 1

Uppgiften går ut på att använda sig utav entity framework tillsammans med MVC för att skapa en hemsida som kan hantera en databasuppkoppling. Det ska vara byggt med code first och scaffholding där scaffholding skapar fullt fungerat CRUD på databasen.

Specifikt för uppgiften:

- Skapa applikation genom MVC
- Använda entity framework core
- Code first sen skapa databas
- Fullt fungerande CRUD 
- Kunna få upp all information (alla böcker), men även kunna se specifik information (om en bok)
- Förstå hur man navigerar och modifierar data i en DBMS
- Första hur DbContext fungerar och hur man ärver klasser från den.
